teli

teli is a lua chatting server hosted on a Raspberry Pi 5 and it was made in about a week in the 2025 winter break. It works by sending a full "terminal screen" of the chat and the TUI. It uses lua-socket to manage the tcp connections between the server and the clients. These are the features implemented:

  • Account management
    • Create
    • Change password
    • Delete
    • Customize profile and name color
  • Moderation
    • Ban accounts
    • Delete accounts
    • Stop server remotely
  • Global chat
  • Private chat (No groups)
  • Message logs for every chat

There is no need to download anything, you connect to the server using netcat in your terminal or any other tcp connection app.
